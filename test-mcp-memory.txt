# Memory MCP Server Test

This file contains test commands to verify that the Memory MCP server is properly installed and functioning. After restarting Cursor to activate the Memory MCP server, run these commands one by one to test its functionality.

## Step 1: Create a test entity

<use_mcp_tool>
<server_name>github.com/modelcontextprotocol/servers/tree/main/src/memory</server_name>
<tool_name>create_entities</tool_name>
<arguments>
{
  "entities": [
    {
      "name": "User_Ibrahim",
      "entityType": "person",
      "observations": ["Uses Cursor as code editor", "Is working on GRC site project", "Located in Hawaii"]
    }
  ]
}
</arguments>
</use_mcp_tool>
